# Proyecto de titulación - Tecnólogo CETI Colomos


## Scripts
   - `yarn start`
   - `yarn test`
   - `yarn build`
   - `yarn eject`

## Stack
 - ⚛️ React.js
 - 💾 Redux
 - ⚡ Chakra UI

 
 ## Instalation
 - Clone repository
 - Run yarn install
 - Create .env file and add REACT_APP_API_URL and specify api url
 - Run yarn start
